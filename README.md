## 介绍

**一个轻量的RPG药水镶嵌解决方案**

FlyBuff 是 Fly系列插件的一员，FlyBuff是一个面向公众的免费插件 且代码开源

本插件所实现的功能 在各大RPG服务器中均有出现 但是我们是免费的
